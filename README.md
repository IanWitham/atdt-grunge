# At the Drive-Thru Podcast website

Site designed and implemented by me.

[See it running here.](https://9k2b3qgkkh.execute-api.ap-southeast-2.amazonaws.com/)

Tech stack:

- ReactJS / Remix (Grunge Stack)
- Tailwind CSS
- [Architect](https://arc.codes) AWS deployment framework

Episode data is pulled hourly from a LibSyn RSS feed by a scheduled AWS Lambda function.

The site itself runs serverless on Lambda.
